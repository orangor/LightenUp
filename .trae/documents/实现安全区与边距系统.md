# 安全区与边距系统实现计划

## 1. 数据结构扩展
- **目标**: 在 `useFabricCanvas` 状态或 `LayerManager` 中管理安全区配置。
- **方案**:
  - 由于安全区是 Canvas 级别的属性，建议在 `LayerManager` 或 `useFabricCanvas` 中维护。考虑到后续序列化，放在 `LayerManager` 似乎更合适，但目前 `CanvasLayer` 只管理图层。为了简单起见，我们将在 `useFabricCanvas` 中引入 `safeArea` 状态，并暴露给外部。
  - 数据结构:
    ```typescript
    export interface SafeAreaConfig {
      enabled: boolean
      top: number
      right: number
      bottom: number
      left: number
      visible: boolean // 控制显示/隐藏
    }
    ```

## 2. 渲染安全区
- **目标**: 在 Canvas 上绘制虚线框表示安全区。
- **实现**:
  - 创建一个专门的 `drawSafeArea` 函数（类似 `drawGuides`）。
  - 使用 `fabric.Rect` 绘制，设置 `strokeDashArray` 为虚线，`fill` 为透明，`stroke` 为警示色（如橙色或灰色）。
  - 该 Rect 应具备属性: `selectable: false`, `evented: false`, `excludeFromExport: true` (确保不被导出)。
  - 需要在 Canvas 尺寸变化 (`resizeCanvas`) 或配置变化时重绘。

## 3. 交互逻辑（吸附与提示）
- **目标**: 拖拽对象时吸附到安全区边界，并显示提示。
- **实现**:
  - 修改 `canvas/utils/snapGuides.ts` 中的 `computeSnap` 和 `computeMeasurements`。
  - 在 `computeSnap` 中，将安全区的四条边加入 `candidatesX` 和 `candidatesY`。
    - X轴: `left`, `width - right`
    - Y轴: `top`, `height - bottom`
  - 视觉反馈: 当吸附到安全区时，可以将安全区边框高亮（或者重用现有的 Guide Line 机制，当吸附源是 'safe-area' 时显示特定颜色）。

## 4. UI 配置面板
- **目标**: 提供界面供用户设置边距。
- **实现**:
  - 在 `RightPanel`（当未选中对象时）或新增 `CanvasSettingPanel` 中添加安全区设置。
  - 提供 Switch 开关 (`enabled`)。
  - 提供 InputNumber 输入四边边距。
  - 提供“统一设置”按钮。
  - 提供“预设比例”按钮 (5%, 10%)。

## 5. 序列化支持
- **目标**: 保存画布时包含 safeArea 配置。
- **实现**:
  - `serialize` 函数需返回 safeArea 配置。
  - `loadFromJSON` 需恢复配置。
  - 这需要修改 `LayerManager` 的 `serialize` 签名，或者新增一个 `getCanvasData()` 方法来获取包括 layers 和 canvas settings 的完整数据。目前 `serialize` 只返回 layers 数组。建议扩展为返回 `{ layers: [], safeArea: ... }`，但这会破坏现有结构。为了兼容，可以将 safeArea 作为特殊图层处理，或者（推荐）修改 `serialize` 返回更完整的 `ProjectData` 结构。鉴于现有架构，我们可以在 `LayerManager` 中增加 `safeArea` 属性，并在序列化时包含它。

## 实施步骤
1.  **utils/safeArea.ts**: 创建安全区绘制与计算工具函数。
2.  **useFabricCanvas.ts**:
    -   引入 `safeArea` 状态。
    -   在 `useEffect` 中监听 safeArea 变化并重绘。
    -   在 `resizeCanvas` 中更新安全区。
    -   传递安全区参数给 `computeSnap`。
3.  **snapGuides.ts**: 更新吸附逻辑，支持传入安全区矩形作为吸附源。
4.  **RightPanel.tsx**: 添加安全区配置 UI（当无选中对象或选中 Canvas 时）。
5.  **LayerManager.ts**: (可选) 集成安全区数据以支持序列化。

Let's go.
