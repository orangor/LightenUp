## 目标
- 在单选模式下（仅选中一个对象），也显示“对齐”分组。
- 单选模式下，对齐参照只能是“参考画布”，隐藏其他选项。
- 保持多选模式下的完整对齐功能（参考选择边界/画布/首个对象）。

## 变更方案
- 修改组件：[RightPanel.tsx](file:///e:/AiDome/momo/daydo/src/ai-image-editor/components/panels/RightPanel.tsx)
- 提取“对齐”分组渲染逻辑为独立子组件 `AlignGroup`，复用于单选与多选视图。
- 在 `AlignGroup` 中根据 `isSelection`（多选）或 `!isSelection`（单选）控制参考下拉框的选项：
  - 多选：显示全部三个选项（选择边界/画布/首个对象）。
  - 单选：仅显示“参考画布”，并强制 `refType` 为 `'canvas'`，且下拉框可禁用或仅展示单项。
- 在单选视图的 return JSX 中插入 `AlignGroup`（建议放在“位置与尺寸”之后）。

## 细节
- 单选时，refType 状态虽由父组件管理，但渲染时若为单选应强制传 `'canvas'` 给对齐方法，或者在 Select 中锁定为 `'canvas'`。
- 为简化体验，单选时 Select 可以设为 disabled 且值为 'canvas'，或者直接只渲染一个 Option 并默认选中。
- 现有 `distribute` 分组仅在多选时显示（单选无分布意义）。
