# Trae Skills 中文使用指南

本文档旨在帮助您了解并使用 `e:\AiDome\momo\.trae\Skills` 目录下的各种增强技能（Skills）。这些技能是模块化的扩展包，能为 AI 提供特定领域的专业知识、工作流和工具集成。

## 什么是 Trae Skills？

Trae Skills 就像是给 AI 安装的“增强插件”。通过加载这些技能，AI 不仅能回答通用问题，还能执行特定的专业任务，比如生成符合特定品牌规范的文档、操作 Excel 表格、连接外部应用（如 Slack, GitHub）等。

## 如何使用 Skills？

使用 Skills 非常简单，您通常不需要记忆复杂的命令。**只需在对话中用自然语言描述您的需求**，AI 会自动识别并调用相应的技能。

**基本原则：**
1. **明确意图**：告诉 AI 您想做什么（例如：“帮我分析这个 Excel 文件”、“生成一份更新日志”）。
2. **提供上下文**：如果技能需要处理特定文件，请在对话中提及或上传该文件。
3. **按需互动**：某些技能（如连接外部 App）可能需要您进行一次性的授权或配置，请按照 AI 的提示操作。

## 核心技能详解与使用示例

以下是您环境中已安装的主要技能及其用法：

### 1. 办公与文档处理 (Document Skills)

这组技能赋予了 AI 处理 Office 文档和 PDF 的强大能力。

*   **XLSX (Excel)**:
    *   **功能**：创建、编辑 Excel 表格，使用公式，分析数据，生成图表。
    *   **示例指令**：
        *   “创建一个财务预算表，包含收入和支出的自动汇总公式。”
        *   “分析 `sales_data.xlsx`，计算每月的平均增长率。”
        *   “把这一列的数据格式改为货币格式。”
    *   **注意**：AI 会优先使用 Excel 公式而不是硬编码数值，以保持表格的动态性。

*   **DOCX (Word) & PPTX (PowerPoint)**:
    *   **功能**：生成报告、合同、演示文稿，解析文档内容。
    *   **示例指令**：
        *   “根据这份会议记录，生成一份 Word 格式的项目周报。”
        *   “把这篇技术文档转换成一个 10 页的 PPT 演示文稿。”

*   **PDF**:
    *   **功能**：读取 PDF 内容，填写表单，提取信息。
    *   **示例指令**：
        *   “从这些 PDF 发票中提取总金额和日期。”
        *   “帮我填写这个 PDF 申请表。”

### 2. 设计与品牌 (Design & Brand)

*   **Brand Guidelines (品牌规范)**:
    *   **功能**：确保生成的文档、图片或设计符合 Anthropic 的官方品牌风格（颜色、字体、排版）。
    *   **示例指令**：
        *   “把这个 PPT 美化一下，应用官方品牌颜色。”
        *   “生成一份符合品牌规范的 HTML 报告。”
    *   **特点**：自动应用主色调（深色/浅色/灰色）和强调色（橙/蓝/绿），以及指定字体（Poppins/Lora）。

*   **Canvas Design**:
    *   **功能**：创建视觉设计、海报或图表。
    *   **示例指令**：
        *   “设计一张关于‘人工智能未来’的活动海报。”

*   **Theme Factory**:
    *   **功能**：为文档或网页应用预设的主题（如“极地霜冻”、“午夜银河”等）。
    *   **示例指令**：
        *   “把这个报告的配色改成‘Modern Minimalist’风格。”

### 3. 外部连接 (Connect Apps)

*   **功能**：通过 Composio 平台连接 1000+ 外部应用，如 GitHub, Slack, Gmail, Notion 等。
*   **初次使用**：
    1. 在对话中输入 `/plugin install composio-toolrouter` 安装插件。
    2. 输入 `/composio-toolrouter:setup` 进行配置和授权。
*   **示例指令**：
    *   “给 `user@example.com` 发一封邮件，主题是‘项目更新’。”
    *   “在 GitHub 上创建一个 issue，标题是‘修复登录 Bug’。”
    *   “把今天的会议纪要发送到 Slack 的 #general 频道。”

### 4. 开发者工具 (Developer Tools)

*   **Changelog Generator**:
    *   **功能**：分析 Git 提交记录，自动生成用户友好的更新日志。
    *   **示例指令**：
        *   “根据最近一周的 commit 生成一份 Release Notes。”

*   **Webapp Testing**:
    *   **功能**：使用 Playwright 测试 Web 应用。
    *   **示例指令**：
        *   “帮我测试一下登录页面的表单验证功能。”

*   **Mcp Builder**:
    *   **功能**：辅助构建 MCP (Model Context Protocol) 服务器，用于扩展 AI 的工具集。

### 5. 实用工具 (Utilities)

*   **File Organizer**:
    *   **功能**：智能整理混乱的文件和文件夹。
    *   **示例指令**：
        *   “整理一下 `Downloads` 文件夹，把文件按类型归档。”

*   **Slack Gif Creator**:
    *   **功能**：制作适合 Slack 使用的动图（GIF）。
    *   **示例指令**：
        *   “做一个‘High Five’的 GIF 动图，要用品牌色。”

*   **Internal Comms**:
    *   **功能**：撰写各类内部沟通文档（周报、公告、FAQ）。
    *   **示例指令**：
        *   “写一份关于系统升级的内部通知邮件。”

*   **Tailored Resume Generator**:
    *   **功能**：根据职位描述定制简历。
    *   **示例指令**：
        *   “根据这个 JD，优化我的简历内容。”

## 进阶：如何创建自己的 Skill？

如果您想扩展 AI 的能力，可以使用 **Skill Creator**。

1. **构思**：确定您想让 AI 学会什么新技能（例如：自动化某种特定的代码重构，或者查询公司内部的 API）。
2. **指令**：告诉 AI “我想创建一个新技能”。
3. **流程**：
    *   AI 会使用 `skill-creator` 引导您初始化技能结构。
    *   您需要在 `SKILL.md` 中定义技能名称、描述和指令。
    *   可以添加 Python 脚本或参考文档到技能包中。
    *   最后使用 `package_skill.py` 打包发布。

## 常见问题

*   **问：如何知道某个技能是否生效？**
    *   答：当您发出相关指令时，AI 的回复通常会显示它正在使用某个工具（例如 "Reading file..." 或 "Executing script..."）。您也可以直接问 AI：“你现在可以使用哪些技能？”

*   **问：`connect-apps` 无法使用怎么办？**
    *   答：请检查是否已安装插件。尝试运行 `/plugin install composio-toolrouter`。
