# 批量导出与命名规则系统实施计划

## 1. 基础设施准备
-   **依赖安装**: 安装 `jszip` 用于文件打包。
-   **工具库**: 创建 `naming.ts` 用于处理文件名模板解析。

## 2. 导出核心能力升级 (`exportImage.ts`)
-   **重构 `getExportDataURL`**:
    -   支持格式: `png`, `jpeg`, `webp`。
    -   支持参数: `multiplier` (1x-3x), `quality` (0-1), `enableTransparent` (仅 PNG/WebP), `cropToSafeArea` (基于安全区裁剪)。
    -   实现裁剪逻辑: 利用 fabric `toDataURL` 的 `left`, `top`, `width`, `height` 参数。

## 3. 业务逻辑实现 (`LayerManager` & `BatchExportService`)
-   **扩展 `LayerManager`**:
    -   新增 `renderVariantOffscreen(variant)`: 为了不干扰主画布，理论上可以创建一个临时的 `fabric.StaticCanvas` 来渲染和导出，但这需要完全独立的 DOM 节点。考虑到性能和复杂性，**复用主 Canvas** 是最可行的，但需要做好状态保护。
    -   实现 `exportBatch(variants, config, onProgress)`:
        -   保存当前状态 (`saveState`)。
        -   标记 `isExporting = true` (暂停 notify 以防 UI 抖动)。
        -   串行遍历 variants:
            -   `loadFromJSON` 加载数据。
            -   `resizeCanvas` 调整尺寸。
            -   `renderAll`。
            -   调用 `exportImage` 获取 Blob。
            -   调用 `naming` 生成文件名。
            -   添加到 ZIP。
            -   调用 `onProgress` 更新进度。
        -   恢复原始状态 (`switchVariant(originalId)`).
        -   `isExporting = false`.
        -   返回 ZIP Blob。

## 4. UI 界面开发
-   **新增组件 `ExportModal.tsx`**:
    -   **左侧**: 变体选择列表 (Checkbox Group)，显示尺寸和名称。
    -   **右侧**: 导出配置。
        -   格式 (PNG/JPG/WebP)。
        -   倍率 (1x/2x/3x)。
        -   背景 (透明/填充)。
        -   范围 (全图/安全区)。
        -   命名规则 (输入框 + 预览)。
            -   提供预设 Tags 点击插入 (e.g. `{width}`, `{name}`).
    -   **底部**: 导出按钮 (带 Loading 状态)。

## 5. 集成与测试
-   在 `index.tsx` 引入 `ExportModal`。
-   连接 `onExportAll` 触发 Modal。
-   测试不同格式、倍率、裁剪和批量打包功能。

## 6. 命名规则规范
-   默认模板: `poster_{canvasName}_{width}x{height}.{ext}`
-   支持变量:
    -   `{project}`: 项目名称 (默认为 "Project")
    -   `{canvas}`: 画布名称
    -   `{w}`: 宽
    -   `{h}`: 高
    -   `{date}`: 日期 (YYYYMMDD)
    -   `{index}`: 序号 (01)

## 7. 实施步骤
1.  **Run Command**: 安装 `jszip`。
2.  **Coding**: 创建 `naming.ts` 和更新 `exportImage.ts`。
3.  **Coding**: 在 `LayerManager` 中实现批量导出逻辑。
4.  **Coding**: 创建 `ExportModal` 组件。
5.  **Coding**: 集成到 `index.tsx`。
